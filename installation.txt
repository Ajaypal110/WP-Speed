Installation Instructions for NitroMan

Automatic Installation:
1. Download the plugin as a .zip file.
2. Log into your WordPress admin dashboard.
3. Navigate to Plugins -> Add New.
4. Click on the "Upload Plugin" button at the top of the page.
5. Choose the zip file you downloaded and click "Install Now".
6. After the installation is complete, click "Activate Plugin".
7. You can now access the NitroMan plugin settings from your WordPress dashboard menu.

Manual Installation:
1. Extract the downloaded zip file.
2. Upload the extracted `NitroMan` folder to the `/wp-content/plugins/` directory on your server via FTP or your hosting control panel's file manager.
3. Log into your WordPress dashboard and navigate to the Plugins page.
4. Locate "NitroMan" in the list and click "Activate".
